Finder Address Bar in Mac OS X

One thing that I never liked about Finder is that there is no address bar.

I don't know why the person at Apple chose not to reveal it, since at its core the address is how finder works, Windows uses it, and web browsers use it.

Anyway, found this awesome answer on StackOverflow today.

To go to a folder in finder via typing an address: 

command+shift+g

To show the full path in finder run this on terminal: 

defaults write com.apple.finder _FXShowPosixPathInTitle -bool YES && killall Finder

P.S. Yes I know that links goes to superuser, but it's basically stackoverflow.
