Bash Prompt for Heavy Git Users

git comes with utilities to easily customize your bash prompt to show what branch you are on, whether there are untracked files in that branch, whether there are staged commits, and whether there are stashed files. If this isn't a pain for you yet, don't worry about it.

Steps:

  1. Download git source code:

    git clone https://github.com/git/git.git

  2. Edit your .bash_profile (this script gets run every time you open a new terminal window)

    mate ~/.bash_profile (or subl/vim/etc--whatever you use for text editing)

  3. Add these lines to the top of your bash_profile to set some bash variables to make this git thing show status of the git repo:

    GIT_PS1_SHOWDIRTYSTATE=true GIT_PS1_SHOWUNTRACKEDFILES=true GIT_PS1_SHOWSTASHSTATE=true

  4. Add these lines to import the git prompt tools:

    source ~/git/contrib/completion/git-completion.bash source ~/git/contrib/completion/git-prompt.sh

  5. Update your bash PS1 variable, which is the variable that creates your bash prompt. Add this part to your PS1:

    $(__git_ps1 " %s ")

  6. If you want to copy my PS1, here it is:

    PS1='[\e[0;32m]\u[\e[m] [\e[1;34m]\w[\e[m] [\e[1;32m]$(__git_ps1 " %s ")\$[\e[m] [\e[1;37m]'
