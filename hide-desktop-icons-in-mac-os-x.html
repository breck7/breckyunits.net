Hide Desktop Icons in Mac OS X

I hate having anything on my desktop.

Run these two commands in terminal to hide the icons: 

defaults write com.apple.finder CreateDesktop -bool false
killall Finder

Thanks to http://osxdaily.com/2009/09/23/hide-all-desktop-icons-in-mac-os-x/ for the tip!
