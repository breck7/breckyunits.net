Git stash

Have some changes that you don't want to commit, but also don't want to lose? Use git stash 

git stash

That will stash the changes and revert you to your last checkout. Now you can switch branches/etc.

When you are ready to apply those changes again (either to the same or a different branch) : 

git stash apply

To clear the stash: 

git stash clear
